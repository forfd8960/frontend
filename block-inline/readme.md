## Block

### Block level elements

Block Elements begin on a new line and take up full width of the page.

```html
<div></div>
<form></form>

h1 - h6
```


## Inline

Inline elements work within the flow of surrounding content rather than breaking onto their own line.

### Inline elements

```html
<a href=""></a>

<img src="">

<input>
<label></label>
<b></b>

<em>
<span>
```
